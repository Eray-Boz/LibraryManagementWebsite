## Project Features
This project offers a comprehensive library management system. Its main features are:

### 1. Book Management (Book CRUD)
- **Add Book:** New books can be added to the library. (Title, Author, ISBN, Publication Year, Category, etc.)
- **Edit Book:** Existing book information can be updated.
- **Delete Book:** Book records can be deleted from the system.
- **List Books:** All books can be viewed and searched.

### 2. Member Management (Member CRUD)
- **Add Member:** New members can be registered to the library.
- **Edit Member:** Member information can be updated.
- **Delete Member:** Membership records can be deleted.
- **List Members:** Existing members can be listed.

### 3. Borrowing and Return (Loan Management)
- **Borrow Book:** Members can borrow books.
- **Return Book:** Returned books can be processed.
- **Tracking:** Who has which book and return statuses can be tracked.
